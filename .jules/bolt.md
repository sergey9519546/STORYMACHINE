## 2024-05-16 - Memoization of Synchronous Parsing in React **Learning:** Heavy string parsing functions running multiple times per render cycle in inline JSX can severely degrade performance on high-frequency input components like Editors. **Action:** Enforce `useMemo` for derived state from props, especially expensive O(N) operations like text parsing (`parseFountain`), and reuse the computed result instead of invoking the parsing function repeatedly during render.
