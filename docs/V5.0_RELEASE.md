## 🏗️ Architecture Overview

### **Event-Sourced Kernel**
Immutable append-only narrative log with cryptographic integrity:
- **Dual Temporal Dimensions:** Story-time (diegetic) + presentation-time (audience reveal)
- **Reality Layers:** Diegetic, dream, memory, hypothetical, deceptive
- **Timeline Branching:** Fork/merge for "what-if" exploration
- **Cryptographic Chain:** SHA-256 hashing like Git commits
- **Backward Compatible:** Adapters for existing StoryCommit system

### **Trinity Verification Gate**
Three-layer verification system (all must pass):

**Layer 1: Story Graph Verifier**
- Promise-payment ratio ≥ 70%
- Forward causality ≥ 60%
- Arc coherence validation
- Escalation monotonicity

**Layer 2: OWNE Verifier**
- World invariants (fact consistency)
- Intentional planning (character motivation)
- Promise/payoff logic (setup distance)
- Temporal feasibility (STNU constraints)

**Layer 3: Pre-Flight Auditor**
- Epistemic consistency (knowledge paths)
- Possession tracking (object custody)
- Spatial feasibility (travel constraints)

### **Quantum Story Field**
Superposition-based narrative exploration:
- Maintains 100-1000 parallel story-states
- Entanglement tracking (setup→payoff dependencies)
- Probability weighting (Story Graph metrics)
- Wavefunction collapse (user selection)
- Automatic dependency propagation

## 🚀 Usage Examples

### **Basic Event Sourcing**
```typescript
import { createEventStore } from './server/nvm/kernel';

const store = createEventStore();

const event = store.append({
  op: { op: 'ADD_FACT', fact: { /* ... */ } },
  storyTime: 100,
  presentationIndex: 0,
  derivedFrom: [],
  createdBy: 'user_authored',
  realityLayer: 'diegetic',
  sceneIdx: 0,
  assertions: [],
  parentHash: null,
});

// Query by story-time (enables flashbacks)
const events = store.getEventsBeforeStoryTime(150);

// Get state snapshot
const state = await store.snapshot({ storyTime: 200 });
```

### **Trinity Gate Verification**
```typescript
import { runTrinityGate } from './server/nvm/kernel';

const verification = await runTrinityGate(
  proposedEvent,
  currentState,
  allEvents
);

if (!verification.pass) {
  console.log('Violations:', verification.violations);
  console.log('Repairs:', verification.repairSuggestions);
  // Block commit - plot hole prevented!
}
```

### **Quantum Story Exploration**
```typescript
import { createQuantumField } from './server/nvm/quantum';

const field = createQuantumField(eventStore);
const rootState = await field.initialize(eventStore);

// Create branches
const branch1 = await field.branch(rootState.stateId, [option1]);
const branch2 = await field.branch(rootState.stateId, [option2]);

// Get top probable states
const topStates = field.getTopStates(5);

// User selects one
const result = field.collapse(topStates[0].stateId);
console.log(`Resolved ${result.entanglementChanges.resolvedSetups} setups`);
```
