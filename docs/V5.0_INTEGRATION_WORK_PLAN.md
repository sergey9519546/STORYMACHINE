### Task 2.2: Implement Dual-Write in Stage
**Owner:** Senior Developer  
**Duration:** 6 hours

**Actions:**
1. Modify `server/engine/Stage.ts`
2. Add EventStore instance
3. Implement dual-write on commit:
   ```typescript
   async commit(storyCommit: StoryCommit) {
     // Existing commit logic
     await this.legacyCommit(storyCommit);
     
     // V5 dual-write (if enabled)
     if (V5_FEATURES.dualWrite) {
       const events = commitsToEvents([storyCommit]);
       await this.eventStore.appendBatch(events);
     }
   }
   ```
4. Add error handling and logging

**Deliverables:**
- Updated `Stage.ts` (100 LOC changes)
- Dual-write tests (20+ tests)
- Error handling for dual-write failures

**Success Criteria:**
- [ ] Stage writes to both systems

- [ ] Performance impact <5ms per commit
