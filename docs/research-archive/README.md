# Research Archive

Vendored copies of the surviving canonical-shelf documents from the
`MAIN_StoryMachine_Engine_Logic` research folder, per
`docs/research-audit/MASTER_RESEARCH_AUDIT.md` (2026-07-10 audit: six
subagents read all ~130 files, 68 superseded/duplicate files were moved to
`_superseded_2026-07-10/`, 67 keepers remained). This directory vendors the
text-format survivors that existed at the folder's top level at the time of
vendoring, so the research canon travels with the repo instead of living
only in an external, sync-fragile OneDrive folder.

> **Every AI-generated research doc here contains fabricated citations
> somewhere — adopt mechanisms, never citations; `v35_integration_plan.md`
> is the correction reference.**

## Provenance

Vendored from `MAIN_StoryMachine_Engine_Logic/` (top level only; the
`_superseded_2026-07-10/` subfolder and the `STORYMACHINE V1 REPO` subfolder
were excluded, as were archives/binaries — zips, `.gz`, `.skill` files).
Line endings normalized to LF; content otherwise copied byte-for-byte from
the source `.md`/`.txt` files. No `.docx` or `.pdf` survivors existed at the
source folder's top level at vendoring time, so none were converted.
